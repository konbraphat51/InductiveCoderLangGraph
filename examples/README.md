# Examples

This directory contains example data and prompts to demonstrate the inductive coding tool.

## Example: Customer Review Analysis

### Data Files

The `data/` directory contains three sample customer reviews:
- `review1.txt`: Mixed review with positive service but negative wait times
- `review2.txt`: Positive review highlighting quality and support
- `review3.txt`: Negative review about product quality and service

### Running the Example

#### Coding Mode (Sentence-level)

```bash
# From the project root
uv run inductive-coder analyze \
  --mode coding \
  --input-dir ./examples/data \
  --prompt-file ./examples/example_prompt.md \
  --output-dir ./examples/output_coding
```

This will:
1. Read all review files
2. Create a code book based on the themes found
3. Apply codes to individual sentences
4. Save results to `./examples/output_coding`

#### Categorization Mode (Document-level)

```bash
uv run inductive-coder analyze \
  --mode categorization \
  --input-dir ./examples/data \
  --prompt-file ./examples/example_prompt.md \
  --output-dir ./examples/output_categorization
```

This will:
1. Read all review files
2. Create a code book based on the themes found
3. Apply codes to entire documents
4. Save results to `./examples/output_categorization`

### Viewing Results

After running the analysis, view the interactive UI:

```bash
# For coding mode results
uv run inductive-coder ui --results-dir ./examples/output_coding

# For categorization mode results
uv run inductive-coder ui --results-dir ./examples/output_categorization
```

### Expected Output

The analysis should identify codes such as:
- **Positive Service**: Friendly staff, helpful support
- **Negative Service**: Unhelpful service, poor support
- **Product Quality**: Quality satisfaction or dissatisfaction
- **Logistics Issues**: Wait times, shipping, returns
- **Price/Value**: Pricing concerns, value perception
- **Repeat Purchase**: Willingness or unwillingness to return

The exact codes will be generated by the LLM based on the data and your prompt.
